# nginx-alias与root区别

## 如果配置项为alias

```text
location /img/ {
    alias /var/www/image/;
}
```

> 若按照上述配置的话，则访问/img/目录里面的文件时，ningx会去/var/www/image/目录找文件

## 如果配置项为root

```text
location /img/ {
    root /var/www/image;
}
```

> 若按照上述配置的话，则访问/img/目录里面的文件时，ningx会去/var/www/image/img/目录找文件
